Built a speech-based digit recognition system from scratch in C++ using Hidden Markov Models and a custom-recorded dataset. The project covered the full pipeline from recording and preprocessing speech signals to training and evaluating sequence models.

<ul class="tag-list">
  <li>Speech Processing</li>
  <li>C++</li>
  <li>HMM</li>
</ul>

* Implemented the recognition logic from first principles rather than using a high-level speech library.
* Worked with custom audio samples, feature extraction, sequence modeling, and digit-level classification.
* Built as a course project at IIT Guwahati.
