Implemented standard deep reinforcement learning algorithms from scratch using PyTorch and evaluated them across robotic and control environments. This project strengthened my foundations in value-based learning, actor-critic methods, replay buffers, exploration, and continuous control.

<ul class="tag-list">
  <li>Deep RL</li>
  <li>PyTorch</li>
  <li>Webots</li>
  <li>Gymnasium</li>
</ul>

* Implemented DQN, Double DQN, prioritized experience replay, DDPG, and SAC.
* Tested learning behavior in Webots, Gymnasium, and custom simulation pipelines.
* Explored distributed learning setups and robotic navigation tasks.
* Used these implementations as a base for understanding federated and multi-agent reinforcement learning behavior.
* Studied how replay design, target updates, and exploration choices affect stability in robotic tasks.
